# Meep Industries — Concert Photography Portfolio Site

## Purpose
A minimal, fast portfolio + licensing site for concert photography.

## Goals
- Showcase the library
- Self-serve licensing (editorial/commercial tiers)
- Drive traffic from Google image search + music press

## Stack (tentative)
- Static site (Astro or plain HTML/CSS) — zero hosting cost on GitHub Pages / Cloudflare Pages
- Optional: Stripe for self-serve licensing payments

## Sections
- Home / gallery
- About / artist info
- Licensing page (what's available, how to license, pricing)
- Contact
